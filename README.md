### ✒️ 프로젝트 목표
1. 첫 팀프로젝트인 해당 프로젝트를 통한 팀원과의 협업 능력 향상
2. REACT를 활용한 UI/UX 구현으로 SPA방식으로 빠른 UI 렌더링과 반응성 구현하기
3. 데이터 CRUD와 다양한 이벤트 처리 기능을 구현하여 비즈니스 로직 학습하기
4. MySQL을 활용한 데이터 관리
5. Client와 Server, DB간의 연동 경험

## 📌 주요 기능
✔️ 로그인<br>
- 로그인 시 로컬스토리지 생성<br>
- 토큰으로 암호화 (jsonwebtoken)<br>
- 유효성 체크 (필요 정보 입력)<br>
- 아이디 찾기<br>
- 비밀번호 변경<br>


<br>

✔️ 회원가입 & SNS회원가입 <br>
- 유효성 체크 (필요 정보 입력, 약관동의 입력)<br>
- 네이버 API <br>
- 카카오 API<br>
<br>

✔️ 마이 페이지<br>
- 회원정보 조회 및 수정<br>
- 우편번호 검색 기능 (react-daum-postcode)<br>
- 나의 주문내역 날짜별 조회<br>
- 관심상품 내역 조회<br>
- 작성한 리뷰 내역 조회<br>
- 배송지 삭제 및 추가<br>
- 로그아웃, 회원탈퇴<br>
- 페이지네이션 기능 구현<br>

<br>

✔️ 메인 페이지<br>
- 베스트 상품 이미지 슬라이드 (swiper)<br>
- DB연동 하여 제품 정보 조회 후 상품 출력<br>

<br>

✔️ 장바구니<br>
- 장바구니에 담은 상품 목록 출력<br>
- 상품별 개별체크 및 모두선택 기능 구현<br>
- 선택된 상품의 개수 및 가격에 따른 총가격 표시<br>

<br>

✔️ 결제페이지<br>
- 우편번호 검색 기능 (react-daum-postcode)<br>
- 고객정보 입력 시 유효성 체크<br>

<br>

✔️ 상품 전체페이지<br>
- 페이지네이션 사용<br>
- 카테고리 선택 시 해당되는 상품만 출력<br>

<br>

✔️ 상품 상세페이지<br>
- 상품명, 리뷰개수, 가격정보, 상품개수에 따른 총가격 출력<br>
- 상품리뷰 목록 조회, 작성, 삭제, 수정(CRUD), 사진 등록 (multer)<br>

<br>

✔️ 검색페이지<br>
- 검색 키워드 입력 시 해당 상품 목록 출력<br>

<br>

✔️ 관리자페이지<br>
- 상품 등록 (multer)<br>
